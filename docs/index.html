Core_Features

[Declarative]

Syntax_Agnostic

Describe game content logically. Say goodbye to boilerplate code and unreadable dictionaries.

[Speed]

Native_execution

Compiled in pure C++. Zero runtime deps. It parses thousands of nodes before you can blink.

[Safe]

Strict_validation

Catches typos, broken references, and logic errors at compile time, not at play time.

[AI_Hooks]

LLM_ready

Built-in utilities to translate `.lex` files into agentic memory representations via queries.

[Custom]

Schema_freedom

Tower defense, RPG, visual novels. Use `--types` to define exactly the node types you need.

[Raw]

Single_Binary

No pip, no npm, no nightmare setups. One executable to rule your entire content pipeline.
