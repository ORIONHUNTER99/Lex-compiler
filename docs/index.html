Simple, Readable Syntax

game.lex
structure Turret {
    name: "Basic Turret"
    cost: { Gold: 50 }
    damage: 10
    range: 2
}

structure Cannon {
    name: "Heavy Cannon"
    cost: { Gold: 100, Steel: 20 }
    damage: 25
    range: 3
    
    available_if has_technology(Gunpowder) {}
}
lexc game.lex -t lua,json,gd,cs

Features

🎯 Declarative Syntax

Describe what, not how. Clean, readable definitions that anyone can understand.

🔄 Multi-Target Output

Write once, compile to Lua, JSON, GDScript, C#, LÖVE2D, or Defold with one command.

🎮 Engine Agnostic

Use built-in Imperium schema or define your own types for any game genre.

✅ Semantic Validation

Catch errors before runtime with type checking and cross-reference validation.

🤖 AI-Ready

Generate context for LLMs with --context flag. Query your data in natural language.

⚡ Fast & Lightweight

Native C++ compiler. No dependencies. Compiles in milliseconds.
